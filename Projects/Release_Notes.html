Release Notes for X-CUBE-BLEMGR Projects

Copyright © 2024 STMicroelectronics

Purpose

The X-CUBE-BLEMGR projects provide applications to help the user to set and configure bluetooth connectivity from board to mobile application such as ST BLE Sensor.
